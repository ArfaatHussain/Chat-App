A **Realtime One-to-One Chat Application** built as a semester project using **React Native** and **Firebase**. This mobile app allows users to register, log in, add contacts, and chat in real time. The app was built collaboratively by two developers with a focus on learning realtime features and Firebase integration.

## Install Dependencies

Use the package manager npm to install foobar.

```bash
npm install
```

## Run the App
```bash
npx expo start
```
